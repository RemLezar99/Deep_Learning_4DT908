# Probability and Information Theory (Python Implementation)

## Setup Instructions

### 1. Create virtual environment

```bash
python3 -m venv venv
```

### 2. Activate environment

**macOS/Linux**

```bash
source venv/bin/activate
```

**Windows**

```powershell
venv\Scripts\Activate.ps1
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Then open:

```
notebooks/probability_information_theory.ipynb
```

## Libraries Used

* numpy
* pandas
* matplotlib
* scipy
* seaborn (optional)
* scikit-learn (optional)

## Notes

* The epidemic data generation step from MATLAB is skipped.
* The provided dataset `epidemic_process.csv` is used directly.
* All MATLAB code has been translated into Python.
